= Installation =

Just add ScaleMM2.pas as the first unit in your Delphi project file (.dpr).
--------------------
program <YourApp>;

uses
  ScaleMM2 in '<yourpathscalemmpath>\ScaleMM2.pas',
  <etc> 
--------------------
Note: also add the <yourpathscalemmpath> directory to your search path!
